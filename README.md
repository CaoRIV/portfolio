# Personal Story Portfolio

An interactive personal portfolio built with React, Vite, and Framer Motion. The project presents a personal profile through a polished one-page experience that combines storytelling, visual rhythm, and smooth motion.

The site is designed as a guided portfolio narrative rather than a static resume page. It introduces the person, explains their working style, highlights skill areas, presents featured projects, shows career or learning milestones, and ends with a focused contact section.

## Overview

This portfolio is structured around a clear personal story. The first screen introduces the name, role, location, and core positioning, supported by an abstract interactive stage that represents the working process. The following sections expand that story through traits, skills, project cards, timeline entries, and a contact form.

The layout is responsive and adapts from wide desktop screens to mobile devices while keeping the main content readable and visually balanced.

## Main Sections

- Hero section with name, role, tagline, location, call-to-action buttons, and an animated portfolio system visual.
- About section that describes the working process and personal approach.
- Traits section for key professional qualities.
- Skills section grouped into product, frontend, and craft-related capabilities.
- Featured work section for selected projects, roles, stack details, and future case study links.
- Timeline section for milestones, experience, or education.
- Contact section with a simple form and social/contact links.

## Design Direction

The visual style uses a warm editorial palette, large expressive typography, soft surfaces, subtle grain, and layered panels. The UI takes inspiration from modern product storytelling sites, with a focus on calm spacing, tactile depth, and motion that supports the content instead of distracting from it.

Interactive details such as hover states, scroll reveals, animated cards, focus styles, and form feedback help the page feel more complete and polished.

## Tech Stack

- React for the component structure.
- Vite for the frontend build setup.
- Framer Motion for page entrance animations, scroll-based movement, and subtle looping motion.
- Plain CSS for layout, responsive styling, visual surfaces, and interaction states.

## Current State

The portfolio currently uses placeholder profile data, project titles, role labels, social links, and contact information. These values are organized in the profile object inside the main React file so the content can be customized without changing the overall layout.

The project already includes a `.gitignore` for local dependencies, build output, logs, and test artifacts.
