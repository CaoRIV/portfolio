# Portfolio

A personal portfolio built with React and Vite. Features an interactive design with smooth animations and a responsive layout that works on all devices.

## Overview

This portfolio presents a personal story through an engaging, interactive experience. The design is clean and modern, with a focus on visual storytelling and smooth user interactions.

## Features

- **Hero Section** – Introduction with name, location, and call-to-action buttons
- **About Section** – Overview of working approach and interests
- **Skills Section** – Key technical skills
- **Projects Section** – Featured work and projects
- **Timeline** – Learning journey and milestones
- **Contact Section** – Email, social links, and contact form

## Design

The site uses:
- Warm, editorial color palette
- Large, expressive typography
- Smooth animations and hover effects
- Fully responsive design for desktop and mobile
- Modern product storytelling inspiration

## Tech Stack

- **React** – Component structure
- **Vite** – Fast frontend build tool
- **Framer Motion** – Smooth animations and scroll effects
- **CSS** – Layout and styling

## Getting Started

Clone the repository and install dependencies:

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```
